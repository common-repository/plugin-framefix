Du nutzt eine Domain die als FRame-Weiterleitung dient und kannst wp-admin nicht betreten, bekommst stattdessen eine weiße Seite? Dieses Plugin fixxt das Problem!

== Description ==
Mit diesem Plugin werden zwei Funktionen der Wordpress-Core deaktiviert. Diese zwei Funktionen senden beim Aufruf der Seite diverse HTTP-Informationen mit die in vielen Webbrowsern über Frame-Domains nicht intepretiert werden.
        
== Installation ==

Downloade das Plugin oder installiere es direkt über der Wordpress-installation bzw. der Plugin-Store. Nachdem das Plugin aktiviert wurde, kann es direkt eingesetzt werden.
